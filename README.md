# 2025-richtext-editor

**usage:**

- `❯ git clone https://github.com/hastebrot/2025-richtext-editor`
- `❯ cd 2025-richtext-editor/sources/lexical-react-app/`
- `❯ bun install`
- `❯ bun run dev --port 5678`
- `❯ open -a safari --url "http://localhost:5678/"`
